# A4 색칠 도안 생성기

iPad나 종이에 색칠할 수 있는 A4 세로형 랜덤 도안을 브라우저에서 만드는 정적 웹앱입니다.

## 실행

AI 이미지 생성까지 사용하려면 Node 서버를 띄운 뒤, 웹앱의 `API 설정`에 OpenAI API 키를 입력합니다.

```bash
node server.js
```

그다음 `http://localhost:4174/index.html`로 접속합니다.

API 키는 기본적으로 저장되지 않습니다. `이 브라우저에 API 키 저장`을 켜면 현재 브라우저의 localStorage에 저장됩니다.

## 기능

- 성인용 주제 선택: 랜덤 프리미엄 도안, 코지 인테리어, 보태니컬 인물, 빈티지 마켓, 도시 카페 거리, 판타지 서재, 장식 동물
- OpenAI 이미지 생성 API 기반 AI 컬러링 도안 생성
- 난이도 선택: 가볍게, 집중, 고난도
- 난이도 기준: 가볍게는 큰 색칠 영역과 적은 배경, 집중은 중간 밀도, 고난도는 촘촘한 성인용 디테일로 생성합니다.
- 도안 스타일: 큰 메인 그림, 배경이 있는 장면, 반복 패턴
- 넓은 색칠 여백 옵션
- A4 비율 SVG 미리보기
- PNG 저장, SVG 저장, iPad 공유, 인쇄/PDF
